🌐 เปิดจากเครื่องอื่นใน LAN
เครื่องอื่นในเครือข่ายเดียวกันเปิด URL ด้านบนได้เลย · ต้องเปิด firewall อนุญาตพอร์ต · backend (:4012) เป็น internal ไม่ต้องเปิดสู่เน็ต · แก้ Dashboard ได้ทีละคน (คนอื่นเป็น view จนกว่าจะเตะเข้าแทน)
🔑 API Token (backend ↔ frontend/deploy)
token: —
token "ตายตัว" แชร์ให้ backend+serve+deploy (browser ไม่ถือ) · เปิด/รีเซต = restart backend+frontend+deploy อัตโนมัติ · ป้องกัน การเข้า backend ที่ไม่ผ่าน proxy (เช่น backend เปิดออก LAN) · ยังไม่กัน user ที่เข้าผ่านเว็บ (ใช้ login-gate §33.3 ภายหลัง)
🔐 HTTPS (TLS) — เว็บ frontend + deploy
TLS ใส่ที่ "ขอบ" (เว็บ :frontend/:deploy) · backend/manager คง http ภายใน (proxy ผ่าน TLS ที่ขอบ) · บันทึกแล้ว restart frontend+deploy อัตโนมัติ · self-signed cert browser จะเตือนครั้งแรก (กดยอมรับ/Advanced→Proceed) · domain จริง → ใช้ cert จาก CA/Let's Encrypt · ดูคู่มือ docs/คู่มือ-กู้รหัส-Login.md / handoff §33.2
🔒 ล็อกการเข้า Manager (Access gate)
ต้อง login ก่อนใช้ Manager · แยกจากสิทธิ์ผู้ใช้ (§21) · มีผลครั้งถัดไปที่เปิดหน้านี้ · ตั้งรหัสก่อนเปิดล็อก
